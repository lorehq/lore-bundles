# Lore Marketplace

Community bundles for [Lore](https://github.com/lorehq/lore).

## Bundles

| Bundle | Description |
|--------|-------------|
| [cloudnative-starter](cloudnative-starter/) | Comprehensive development bundle — code review, TDD, architecture, security |
| [stack-skills](stack-skills/) | 5 cognitive firewall skills — adversarial review, creativity sampling, reasoning tracing |
| [maestro](maestro/) | 60+ skills covering language patterns, testing, security, API design, deployment |
| [dev-discipline](dev-discipline/) | Git workflow discipline — commit enforcement, execution planning, multi-agent coordination |

## Install

```bash
lore bundle install <slug>
```
